# DevCamper API

> Backend API for DevCamper application, which ia a bootcamper directory website

## USAGE

Rename "config/config.env.deploy" to "config/config.env" and update the values/settings to your own

## Install Dependencies

```
npm Install
```

## Run application
```
# Run in dev mode
npm run dev

# Run in prod mode
npm start

```
